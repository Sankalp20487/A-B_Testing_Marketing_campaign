A/B Test Overview

Objective:

The purpose of this A/B test is to assess the effectiveness of a digital advertising campaign by determining whether users who were shown marketing advertisements convert at a higher rate than those shown Public Service Announcements (PSAs).

This randomized experiment helps answer the core business question:

Does ad exposure significantly impact user conversion behavior?

Additional Goals:

Beyond evaluating ad effectiveness, this analysis also investigates:

  • When users are most likely to convert by analyzing the day and hour when they saw the most ads.
  • How much exposure influences conversion by exploring the relationship between the number of ads shown and user conversion.

Experimental Design:

Group Description
Test Group Users who were shown ads as part of the marketing campaign (test group = 'ad')
Control Group Users who were shown Public Service Announcements (test group = 'psa')

By comparing conversion rates across these groups, we can quantify the campaigns impact and provide insights for targeting and budget optimization.

Dataset Description

The dataset used in this analysis simulates an A/B marketing campaign conducted by a company to test the effectiveness of advertisements versus Public Service Announcements (PSAs). It contains 588,101 user-level records.

Each row represents a unique user, and the dataset includes the following features:

Column Description
user id Unique identifier for each user (removed during preprocessing)
test group Indicates whether the user saw an ad ('ad') or a PSA ('psa')
converted Boolean flag: True if the user made a purchase, False otherwise
total ads Total number of ads shown to the user
most ads day Day of the week the user saw the most ads
most ads hour Hour of the day (023) the user saw the most ads

During preprocessing, an additional column ads time was created to convert most ads hour into 12-hour format for better interpretability.

Statistical Testing

Hypotheses & Test Metrics

Primary Metric:

Conversion Rate = Number of users who converted / Total users in group

Hypotheses

1. Primary Hypothesis: Does showing ads lead to more conversions than PSAs?

  • Null Hypothesis (H):
    There is no difference in conversion rate between the test and control group.
    $$ P_{\text{ad}} = P_{\text{psa}} $$

  • Alternative Hypothesis (H):
    There is a difference in conversion rate.
    $$ P_{\text{ad}} \ne P_{\text{psa}} $$

2. Secondary Hypotheses (Exploratory):

  • Time of Ad Exposure (most ads day / ads time)
    H: Conversion rate is independent of day/time
    H: Conversion rate depends on day/time

  • Ad Volume (total ads)
    H: Converted and non-converted users saw the same number of ads
    H: Users who converted saw significantly more ads

Tests Used:

Test Purpose
Chi-Squared Test Test categorical association (e.g., test group vs converted)
Shapiro-Wilk Test Test normality of ad exposure per group
Levenes Test Test for equal variance of ad counts
MannWhitney U Test Test if converted users saw more ads (non-parametric)

First we'll pair categorical features with Converted

  • For comparing two categorical features, we use Chi-square test
In[]:
from scipy.stats import chi2_contingency
alpha = 0.05

for variable in df_cat.columns:
    if variable != 'converted':
        # Create a contingency table (cross-tabulation)
        ct = pd.crosstab(df[variable],df['converted'])

        # Perform chi-squared test
        chi2, p, _,_ = chi2_contingency(ct)

        # Display the results
        print(f"\nChi-Squared test for {variable} vs converted:")
        print(f"Chi-Squared value: {chi2}")
        print(f"p-value: {p}")

        # Check for significance
        if p < alpha:
            print(f"The difference in conversion rates across {variable} is statistically significant.")
        else:
            print(f"There is not significant difference in conversion rates across {variable} ")

Now we'll pair numerical feature with Converted

Shapiro Test (For Normality)

  • We'll first check for normality of data and we'll do this by sample i.e we'll segregate the feature total ads into whether its the total number of ads seen by people who have converted and who have not converted.

Levene Test (For equality of variance)

In[26]:
from scipy.stats import shapiro, levene, ttest_ind, mannwhitneyu

# Step 1: Check Assumptions
# Normality assumption
shapiro_stat_true, shapiro_p_value_true = shapiro(df[df['converted'] == True]['total ads'])
shapiro_stat_false, shapiro_p_value_false = shapiro(df[df['converted'] == False]['total ads'])

print(f"ShapiroWilk test for normality (True group): p-value = {shapiro_p_value_true}")
print(f"ShapiroWilk test for normality (False group): p-value = {shapiro_p_value_false}")

# Equality of variances assumption
levene_stat, levene_p_value = levene(df[df['converted']]['total ads'], df[~df['converted']]['total ads'])
print(f"Levene's test for equality of variances: p-value = {levene_p_value}")

ShapiroWilk test for normality (True group): p-value = 1.638680987007771e-98
ShapiroWilk test for normality (False group): p-value = 9.883049430735801e-204
Levene's test for equality of variances: p-value = 0.0
  • Since all the p-values are below 0.05, we conclude:
    1. Data is not normally distributed
    2. Variances are not equal
  • Therefore, we need to choose a non-parametric test.
In[27]:
# Step 2: Perform a Suitable Test
alpha = 0.05

if shapiro_p_value_true > alpha and shapiro_p_value_false > alpha and levene_p_value > alpha:
    # Assumptions met - use t-test for means
    t_stat, t_p_value = ttest_ind(df[df['converted']]['total ads'], df[~df['converted']]['total ads'])
    print(f"Independent two-sample t-test: p-value = {t_p_value}")
else:
    # Assumptions not met - use MannWhitney U test for medians
    u_stat, u_p_value = mannwhitneyu(df[df['converted']]['total ads'], df[~df['converted']]['total ads'])
    print(f"MannWhitney U test: p-value = {u_p_value}")

Actionable Recommendations

Based on the results of the A/B test, the following recommendations can be made to optimize future marketing campaigns:

  1. Prioritize Ad Placement on Mondays:
    Conversion rates were highest on Mondays (~3.3%), suggesting that users are more likely to convert at the start of the week. Increase ad frequency or budget allocation on this day.

  2. Target Peak Conversion Hours (35 PM & 8 PM):
    Analysis of ads time revealed highest conversion between 3 PM to 5 PM and again around 8 PM. These time slots should be prioritized for ad delivery.

  3. Optimize Ad Frequency:
    Users who converted saw more ads than those who didnt (confirmed via MannWhitney U Test). Gradually increasing ad exposure (while avoiding overload) may improve conversion without causing ad fatigue.

  4. Expand A/B Testing to Test Creative or Message Variants:
    Now that the ad format is validated, future tests can explore different ad creatives, copy, or targeting strategies to further improve ROI.

  5. Use PSA as Baseline for ROI Measurement:
    Since the control group was exposed to PSAs, use this setup as a consistent baseline to measure return on future campaigns.
